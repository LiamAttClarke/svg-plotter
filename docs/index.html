svg-plotter

Convert SVG files to GeoJSON

Usage

svg-plotter can be used in both NodeJS and the browser.

Install using npm:

npm install svg-plotter

CLI Usage

Install globally:

 
npm install -g svg-plotter

Example usage

 
svg-plotter ./path/to/input.svg

Print available options

 
svg-plotter --help
